# MQ-001

**问题：** 儿童重症肺炎支原体肺炎，是否可以静脉滴注阿奇霉素？有什么依据？

**标准化查询：** 儿童 重症 肺炎支原体肺炎 静脉注射 阿奇霉素 治疗指南 适应症 疗效
**意图：** IntentType.DETAIL
**回答：** 警示：婴幼儿静脉阿奇霉素需慎重！基于《儿童肺炎支原体肺炎诊疗指南（2023年版）》，重症推荐静脉阿奇霉素10mg/(kg.d)，qd，连用7天，间隔3-4天后第2疗程，总疗程2-3疗程。转换口服时机为病情改善、体温正常。72小时评价疗效。
